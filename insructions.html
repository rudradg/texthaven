How to use Typehaven

Changing text color.

You can pick the colors you want by clicking the color buttons.

Changing the page title.

You can name the title page by clicking this and typing anything, you can not use this for the desktop version.

Downloading your text.

Press download.

Now that you have downloaded it, your file will apear in your directory as a .txt file.

Opening files.

To open a file press "Choose file"

then select a .txt file and press open.

Installing desktop version for Chrome browser.

If not installed you can install it by clicking the TV screen with the arrow pointed down on the top right.
